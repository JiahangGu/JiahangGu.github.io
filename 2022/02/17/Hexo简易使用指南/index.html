hexo的简易使用方式

  1. 安装nodejs
  2. 安装hexo，npm install hexo -g
  3. 安装依赖, npm install –save hexo-deployer-git

hexo命令
hexo new “postName” # 新建文章
hexo new page “pageName” # 新建页面
hexo generate # 生成静态页面至public目录
hexo server # 开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
hexo deploy # 部署到GitHub
hexo help # 查看帮助
hexo version # 查看Hexo的版本
hexo init # 初始化博客，必须在一个空文件夹中

可以在themes下得到next文件夹，并修改_config.yml中theme字段值为next。
./_config.yml是博客配置，更新后需要重新部署，./themes/next/_config.yml是主题配置，更新后要重新生成。

设置文章预览
安装插件
npm install hexo-excerpt –save
在./_config.yml中配置
excerpt:
depth: 5
excerpt_excludes: []
more_excludes: []
hideWholePostExcerpts: true
